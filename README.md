# Cálculo Trabalhista utilizando Bot Telegram

Este projeto é um bot para Telegram desenvolvido em Node.js e Telegraf, desenhado para realizar cálculos de descontos trabalhistas (INSS e IRRF) e benefícios (FGTS) sob o regime CLT (Consolidação das Leis do Trabalho) no Brasil.

Todos os cálculos são baseados nas tabelas oficiais de alíquotas e deduções federais vigentes para o ano de 2024.
